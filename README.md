## Benchmark

```bash
➜  substrate-node-template-benchmark git:(master) ✗ ./target/release/node-template benchmark --chain dev --execution=wasm --wasm-execution=compiled --pallet pallet_template --extrinsic do_something --steps 50 --repeat 20 --template=./.maintain/frame-weight-template.hbs --output=./pallets/template/src/weights.rs
```

## Chain Spec 生成

```bash
# Export the local chain spec to json
./target/release/node-template build-spec --disable-default-bootnode --chain local > ./.chain-spec/customSpec.json
```

chain spec raw:

```bash
./target/release/node-template build-spec --chain=./.chain-spec/customSpec.json --raw --disable-default-bootnode > ./.chain-spec/customSpecRaw.json
```
